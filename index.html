Brains:  Raspberry Pi

The Raspberry Pi has enough capabilities that its best suited for:
- the automated handling, storing, & analysis of data
- bypassing an arduino's ~ 32k Byte sketch size limit by allowing new sketches to be uploaded on command or by a schedule whenever and whereever
- can host local web sites for use by the owner's to monitor and manage their garden or other systems that benefits from this pairing
- can be secured much easier since it is an actual computer
- benefits from running a linux based os so that it can be controlled with a configuration management tool which enables us to help give users a easy and secured baseline for their raspberry pis
- can handle getting and sending data across the internet

Brawn:  Arduino

Ardunios are good at lots of things especially dealing with analog sensors. So arduinos do what they do best with Garduino Pi:
- allowing the use of almost any analog device, sensor, etc
- controlling motors and more with precise control
- endless oppurtunities due to it's flexible nature
- it will allow you to learn a lot about coding and more from frequent use
- excellant way to learn about units, memory management, & thinking like an architect
- have fun with lights, sounds, and sensors to personalize your garden!
